# NxA Secret Cipher Language

**NxA** is an official visual and phonetic substitution cipher language created by **Althaaf** and **Neem**. It was engineered to pass secret, unreadable notes on paper or digital text channels while remaining completely typable on a standard computer keyboard layout.

## 🛠️ Design Philosophy
Unlike standard mathematical shifting codes, NxA relies on a unique blend of character geometry, phonetic homophones across multiple languages, and direct symbolic mapping. 

## 📋 The NxA Alphabet Key
Below is the official specification chart for translating standard English characters into NxA:


| English Letter | NxA Symbol | Visual / Phonetic Rule |
| :---: | :---: | :--- |
| **A** | `/` | Represents the right-side slant of the letter |
| **B** | `>` | Mimics the right-facing curves of a B |
| **C** | `<` | Mimics the open curve of a C |
| **D** | `\|` | Represents the straight vertical back spine |
| **E** | `^` | Mimics the top pointed roof framework |
| **F** | `1` | Vertical line matching the spine of an F |
| **G** | `2` | Sequential number progression |
| **H** | `3` | Sequential number progression |
| **I** | `4` | Sequential number progression |
| **J** | `5` | Sequential number progression |
| **K** | `6` | Sequential number progression |
| **L** | `7` | Sequential number progression |
| **M** | `8` | Visually resembles a lowercase 'm' rotated 90 degrees |
| **N** | `9` | Phonetic wordplay on the German word *nein* (nine) |
| **O** | `0` | Perfect visual matching shape |
| **Q** | `?` | Homophone matching the sound of a "question" |

*(Note: The remaining symbols from the master layout `! - _ ) ( [ ] , .` are reserved for mapping out the remaining variables: P, R, S, T, U, V, W, X, Y, Z).*

## 💬 Official Code Examples
* **Plain Text**: `Neem`
* **NxA Code**: `9--8`
* **How it works**: The number `9` creates the 'N' sound, the `--` string acts as a physical bridge stretching out the long 'ee' vowel sound, and the `8` resolves into the terminal 'm' shape.
